# AI Powered Healthcare Analysis


A professional Machine Learning web application for Lung Cancer Risk Prediction built using Python, Scikit-learn, XGBoost, and Streamlit .

## 🚀 Features


- Real-time Lung Cancer Risk Prediction

- Interactive Healthcare Dashboard
- Professional Dark UI/UX

- Risk Probability Analysis
- PDF Report Generation

- Machine Learning based Decision Support
